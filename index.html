Урок №1 · Консоль и переменные

console.log()

📋 Что это: Команда для вывода в консоль (F12 → Console).

🔍 Зачем: отладка, проверка, понимание кода.

// Примеры вывода
console.log("Привет");             // Строка
console.log(42);                   // Число
console.log(переменная);           // Значение
console.log("Ответ:", результат);  // Несколько значений
!Важно: Без console.log() результат в консоли не увидеть.
const let var (устарел)
🔒 const — нельзя переприсвоить
const a = 5; a = 10; ❌ ошибка
но можно менять объект: user.name = "Мария"; ✅
📦 let — можно изменять
let b = 5; b = 10; ✅
действует внутри блока {}
⚠️ var — "всплывает" (hoisting)
console.log(c); // undefined
не используй в новом коде!
Итог: const по умолчанию, let если нужно менять, var — только legacy.

Урок №2 · Именование + типы

📏 Правила именования: осмысленные имена, camelCase (userAge), английские слова.
🐫 camelCase: userName 📐 PascalCase: UserProfile (классы) 🔠 UPPER_CASE: MAX_COUNT

📊 Типы данных

Простые
string, number, boolean, null, undefined, symbol
Ссылочные
Object, Array, Function
Особенности NaN: console.log('string' * 9); // NaN
typeof NaN → "number" (да, число‑монстр) 
// Проверка типа
typeof "текст"      // "string"
typeof 42           // "number"
typeof null         // "object"  (историческая ошибка!)
typeof [1,2]        // "object"
Array.isArray([1,2]) // true ✅

🗂️ Объект 🆚 Массив

Объект 
const obj = {
  'Anna': 500,
  'Alisa': 800
};
obj.Anna      // 500
obj['Alisa']  // 800
obj.Peter = 300;
📌 ключ → значение
Массив 
let a = ['Anna',500,'Alisa',800];
a[0] // 'Anna'
a[1] // 500
a.push('Новый');
📌 индекс → значение
Для пар имя→значение — объект, для списков — массив. Не делай a['Anna'] с массивом.

Урок №4 · Интерполяция (шаблонные строки)

Что такое? Вставка переменных в строку через ${} и обратные кавычки ``.
Старый способ (конкатенация) 
'https://site.com/' + category + '/' + 4
✨ Новый способ 
`https://site.com/${category}/5`
// Живой пример
const user = 'Pavel';
alert(`Hello, ${user}`); 

const price = 100, count = 5;
console.log(`Итого: ${price * count} рублей`); // 500
Используй обратные кавычки `` (клавиша Ё). Внутри ${} можно писать любой JS‑код!
🏁 Итог: интерполяция делает код чище и понятнее, чем склеивание через +.

Операторы, инкремент, логика (для повторения)

➗ остаток 5 % 2 → 1 == vs === 2*4=='8' true 2*4==='8' false 
let incr = 10;
incr++;      // постфикс
console.log(incr); // 11
console.log(incr++); // 11 (а после 12)
console.log(++incr); // 13
&& (И): true если оба true
|| (ИЛИ): true если хоть один true
const isChecked = true, isClose = false;
console.log(isChecked && isClose); // false
console.log(isChecked || isClose); // true
